# Java 2D Game

A simple 2D top-down game built in Java using basic OOP concepts.  
This project was created while learning and revising object-oriented programming and game development concepts.


## Features
- Player movement in four directions
- Speed Powerup
- Sound effects
- Tile-based map system
- Simple collision handling

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- 2D graphics using Java libraries

## How to Run
1. Open the project in IntelliJ IDEA (or any Java IDE).
2. Navigate to the main game file.
3. Run the main class to start the game.

## Controls
- Arrow keys or WASD: Move player

## Purpose of the Project
This project was built to:
- Practice Java and OOP concepts
- Understand game loops and rendering
- Learn basic 2D game structure
